# Notas de diseño de core-linux

## Elección de Rocky Linux 9
Se utiliza Rocky Linux 9 por su estabilidad, soporte a largo plazo y compatibilidad con herramientas empresariales. Es la base ideal para un servidor central en un entorno realista.

## Filosofía del servidor
Este nodo actúa como servidor principal del laboratorio, concentrando:
- servicios críticos,
- seguridad reforzada,
- almacenamiento gestionado,
- auditoría y backups.

## Objetivos
- Reproducir un servidor Linux empresarial.
- Documentar configuraciones y buenas prácticas.
