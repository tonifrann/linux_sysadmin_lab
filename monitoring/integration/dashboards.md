# Dashboards

Grafana utiliza los siguientes dashboards para visualizar las métricas del laboratorio:

| Dashboard                | ID   | Descripción                       |
|--------------------------|------|-----------------------------------|
| Node Exporter Full       | 1860 | Métricas de CPU, RAM, disco y red |

📸 *Captura: dashboard 1860 funcionando en Grafana*

---

## Organización

El dashboard se encuentra dentro de la carpeta `Monitoring` en Grafana para mantener la vista ordenada.

---

## Notas

- El dashboard utiliza Prometheus como datasource.
- Se ampliará la lista conforme se añadan nuevos dashboards al entorno.
