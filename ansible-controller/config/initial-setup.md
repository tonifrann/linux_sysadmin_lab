# Configuración inicial del entorno

Se documentaran los pasos necesarios que hay que realizar en los servidores que se gestionaran con Ansible.

## 1. Usuario administrador
Todos los servidores Linux donde se utiliza Ansible deben tener configurado un usuario administrador que llamaremos ```admin``` con accesp SSH y privilegios de sudo sin necesidad de escribir la contraseña.
